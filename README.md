## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES Modules)
- **Philosophy:** Modular, Unix-inspired file structure.

## File Structure
```text
.
├── index.html          # Main Entry Point
├── .gitignore          # Git Exclusions
├── README.md           # Documentation
├── public/             # Static Assets (favicon, etc.)
└── src/                # Source Code
    ├── assets/         # Images and Icons
    ├── css/            # Modular Styles
    │   ├── animations.css
    │   ├── layout.css
    │   └── main.css
    └── js/             # Modular Scripts
        ├── main.js
        └── utils.js
```

## Features
- **Animated Hero:** CSS-based clouds and sunbeams.
- **Responsive Design:** Mobile-first approach.
- **Smooth Interactivity:** Native JavaScript scroll handling and intersection observers.
